# گزارش اعتبارسنجی پروژه Angoor

این پروژه پیش از بسته‌بندی با بررسی‌های زیر کنترل شده است:

- ساختار Gradle، Gradle Wrapper و GitHub Actions کامل است.
- فایل HTML نهایی داخل `app/src/main/assets/index.html` قرار گرفته است.
- کد JavaScript داخلی با `node --check` بررسی شده و خطای نحوی ندارد.
- تمام فایل‌های XML منابع و Manifest پارس شده‌اند.
- همه Activity، Receiver و Providerهای معرفی‌شده در Manifest در سورس Java موجود هستند.
- ارتباط HTML با پل بومی اندروید برای آلارم، اثر انگشت، گفتار، ذخیره و اشتراک‌گذاری فایل بررسی شده است.
- آیکون‌های Launcher معمولی، گرد و Adaptive از آیکون نهایی انگور ساخته شده‌اند.
- GitHub Actions ابتدا APK را می‌سازد، گزارش کامل Gradle را در Artifact ذخیره می‌کند و سپس Lint را به‌صورت غیرمسدودکننده اجرا می‌کند.

## محدودیت محیط بررسی

ساخت واقعی پروژه در GitHub Actions انجام و خطاهای قبلی شامل تداخل نسخه‌های Kotlin و ویژگی‌های نامعتبر Style برطرف شده‌اند. بررسی محلی این بسته شامل اعتبارسنجی ساختار، XML، JavaScript، آیکون‌ها و Gradle Wrapper است.
